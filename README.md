# Rick and Morty app for learning NestJS

# DEV

1. Clone Repo
2. Execute

```
yarn install
```

3. You need to have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Up and Run DB

```
docker-compose up -d
```

## Stack

- MongoDB
- Nest
